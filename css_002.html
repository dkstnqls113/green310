박스속성

웹페이지 레이아웃이 가장 중요한 속성

  • 요소크기설정 - width, height
  • 영역설정 - padding
  • 외곽선설정 - border
  • 여백설정 - maegin

작성법

  • 전체적용 - 숫자와 단위를 함께 하나만 작성
  • 각자적용 - 숫자와 단위를 4번 써서 작성한다. 순서는 위,오른쪽,아래,왼쪽 작성시 쉼표를 쓰지않고 띄어쓰기를 통해 작성한다.
  • 세트적용- 위아래/왼쪽오른쪽 숫자와 단위를 두번 써서 작성한다. 앞에가 위아래, 뒤에가 왼쪽오른쪽 각자적용과 마찬가지로 띄어쓰기를 통해 작성한다.
  • 부분적용
    • margin-top : 위쪽여백
    • margin-bottom : 아래쪽여백
    • marbin-left : 왼쪽여백
    • margin-right : 오른쪽여백
    • padding-top : 위쪽영역
    • padding-bottom : 아래쪽영역
    • padding-left : 왼쪽영역
    • padding-right : 오른쪽영역

border 설정 방법

박스의 테두리를 말하며 패딩바깥쪽에 위치한다.
색상,굵기,모양으로 설정한다.

  • border-color : 색상기재
  • border-width : 숫자와 단위
  • border-style
    • none - 선을 없앰
    • solid - 실선
    • dotted - 동그라미점으로 점선만들기
    • dashed - 긴 사각형 점선
    • double - 실선 두줄
    • groove - (액자형) - 테두리가 파인것처럼 그려짐
    • ridge - (액자형) - 테두리가 튀어나온 것처럼 그려짐 - groove 반대스타일
    • inset - (액자형) - 내부요소가 파인것처럼 그려짐
    • outset - (액자형) - 내부요소가 튀어나온 것처럼 그려짐 - inset 반대스타일

작성법 - color,width,style를 한번에 쓸수도 있고 각자 쓸수도 있다. 순번에 상관없음
border : red 2px solid(2px 너비의 빨간색 실선)
